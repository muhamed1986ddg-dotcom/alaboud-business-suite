# ALABOUD Business Suite v15.1 Customer Management

## إدارة العملاء
- زر حذف العميل بعد تأكيدين.
- التأكيد الثاني يتطلب كتابة اسم العميل.
- منع الحذف إذا كان لدى العميل حوالات أو دفعات أو ديون.
- عرض عدد المعاملات التي تمنع الحذف.
- زر أرشفة العميل.
- صفحة للعملاء المؤرشفين.
- استعادة العميل المؤرشف.
- تسجيل الحذف والأرشفة والاستعادة في Audit Log.
- النسخ الاحتياطي التلقائي قبل أي تعديل محفوظ.
