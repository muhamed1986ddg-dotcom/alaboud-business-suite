# ALABOUD Business Suite v14.1 Data Safe

## حماية البيانات
- نسخة تلقائية قبل كل عملية حفظ.
- كتابة آمنة باستخدام ملف مؤقت ثم استبدال ذري.
- استعادة تلقائية من آخر نسخة إذا تلف ملف البيانات.
- الاحتفاظ بآخر 30 نسخة افتراضيًا.
- صفحة لتنزيل البيانات وإنشاء النسخ واستعادتها.
- تحديث التطبيق أو الواجهة لا يستبدل مجلد البيانات.

## شرط مهم في Render
أنشئ Persistent Disk واجعل Mount Path:
`/var/data`

ثم أضف Environment Variable:
`DATA_DIR=/var/data`

بدون قرص دائم، قد تفقد خدمة Render المجانية الملفات المحلية عند إعادة التشغيل أو إعادة النشر.
