# ALABOUD v13 Enterprise Alpha 1

## الملفات
- `ALABOUD_Web_v13_0_0_ALPHA1.zip`
- `ALABOUD_Android_v13_0_0_ALPHA1.zip`

## ترتيب النشر
1. انشر نسخة الويب على مستودع الويب وRender.
2. تأكد من نجاح `Deploy live`.
3. ارفع نسخة Android إلى مستودع Android.
4. افتح Actions وشغّل Build Android v13 Enterprise Alpha 1 APK.
5. حمّل APK واختبر تسجيل الدخول، الحوالات، العملاء، الملفات، واتساب والتنبيهات.

## حالة الاختبار
- Web production build: ناجح.
- Backend syntax: ناجح.
- Android XML/static resources: ناجح.
- APK cloud build: يجب تأكيده في GitHub Actions.
