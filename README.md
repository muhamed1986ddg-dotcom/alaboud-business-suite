# ALABOUD Business Suite v13.0.0-alpha.1

هذه نسخة Enterprise Alpha للاختبار، وليست إصدار إنتاج نهائي.

## الموجود فعليًا
- النظام المالي الحالي وصفحاته.
- واجهة ويب وهاتف سوداء وذهبية.
- نموذج الحوالات بعملات USD وCAD.
- تحديثات الويب تظهر داخل تطبيق Android المبني على WebView.
- مسار Health يعرض رقم الإصدار وقناة Alpha.

## غير مكتمل بعد
- إدارة صلاحيات Enterprise القابلة للتخصيص.
- النسخ الاحتياطي التلقائي المجدول.
- الاستعادة الزمنية الكاملة.
- إصدار Google Play الموقّع.

## Render
Build Command:
`npm run render-build`

Start Command:
`npm start`
