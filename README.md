# AlAboud Business Suite v9.0
تصميم أسود وذهبي كامل للموقع والتطبيق.

Render:
- Build: npm run render-build
- Start: npm start

## إصدار v10.0
- الاعتماد على نفس المشروع الأصلي المرفوع.
- تغيير «لوحة التحكم» إلى «القائمة الرئيسية».
- إضافة زر «العودة إلى القائمة الرئيسية» أعلى جميع الصفحات.
- تمييز الصفحة الحالية باللون الذهبي في القائمة الجانبية.
- الحفاظ على جميع الصفحات والوظائف والبيانات.

## إصدار v11.0 — عرض صفحة واحدة على الهاتف
- عند تشغيل الهاتف تظهر القائمة الرئيسية وحدها.
- عند اختيار قسم تختفي القائمة ويظهر القسم وحده.
- زر ☰ القائمة يعيد إظهار القائمة ويخفي الصفحة.
- زر 🏠 الرئيسية يفتح الصفحة الرئيسية وحدها.
- لا يظهر محتوى الصفحات أسفل القائمة.
- تصميم الكمبيوتر بقي دون تغيير.

## إصدار v11.1
- استبدال كلمة AlAboud بصورة شعار شركة العبود التجارية.
- إظهار الشعار أعلى القائمة على الكمبيوتر والهاتف.
- ضبط حجم الشعار ومنع تمدد عناصر القائمة على شاشة الهاتف.

## إصدار v11.2 — إصلاح الهاتف فقط
- تصغير الشريط العلوي في الهاتف.
- زر القائمة وزر الرئيسية يعيدان إلى قائمة الأزرار الكاملة.
- عند فتح قسم تختفي القائمة وتظهر الصفحة وحدها.
- ضبط عرض البطاقات والإجراءات السريعة لشاشة الهاتف.
- عدم تغيير تصميم الكمبيوتر.

## إصدار v11.3 — إصلاح قائمة الهاتف
- إصلاح اتجاه القائمة ليصبح عموديًا.
- إظهار جميع أزرار القائمة تحت الشعار.
- إزالة الفراغ الأسود الناتج عن خروج الأزرار خارج الشاشة.
- السماح بالتمرير داخل القائمة عند كثرة العناصر.
- عدم تغيير تصميم الكمبيوتر.
