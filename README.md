# ALABOUD Business Suite v13.1 Enterprise Professional

## التحديثات المنفذة
- واجهة هاتف أصغر وأكثر احترافية.
- نقل زر تسجيل الخروج إلى أعلى القائمة.
- نافذة تأكيد قبل تسجيل الخروج.
- زر رجوع الهاتف يغلق القائمة المفتوحة أولًا.
- إضافة الليرة السورية SYP إلى قائمة العملات.
- توضيح أن تحديث SYP يدوي لضمان الدقة.
- عرض CAD بجانب مجاميع العملاء.
- تخفيف تمويه الخلفية عند فتح القائمة.
- تحسين بطاقات العملاء وأزرارها على الهاتف.

## النشر
Build:
`npm run render-build`

Start:
`npm start`
