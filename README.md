# AlAboud Business Suite Enterprise v8.5

## الميزات المجمعة
- مركز تنبيهات 🔔 في لوحة التحكم.
- إشعارات العملاء المتأخرين.
- تصنيف التأخير: 7، 15، 30، 60 يومًا.
- إجمالي عدد ومبالغ العملاء المتأخرين.
- تنبيه انخفاض السيولة.
- تنبيه الحوالات غير المكتملة.
- إرسال تذكير واتساب جاهز بضغطة واحدة.
- إعداد عدد الأيام الذي يبدأ بعده التنبيه.
- إعداد حد انخفاض السيولة.
- قالب رسالة واتساب قابل للتخصيص.
- إجراءات سريعة مناسبة للهاتف.
- العملاء والحوالات والدفعات وكشوف الحساب.
- آخر الأسعار الملونة.
- رأس المال الكلي ودورانه.
- التقرير الشهري والطباعة PDF.

## واتساب
الفتح برسالة جاهزة يعمل بدون اشتراك إضافي.
الإرسال التلقائي الكامل يحتاج WhatsApp Business API رسميًا من Meta أو مزود معتمد.

## Render
Build Command:
npm run render-build

Start Command:
npm start

Health Check:
/api/health


## إصدار v8.6 — صفحة العملاء المتأخرون
- صفحة مستقلة في القائمة الرئيسية.
- شارة بعدد العملاء المتأخرين.
- البحث بالاسم أو رقم الهاتف.
- فلترة حسب أكثر من 7 أو 15 أو 30 أو 60 يومًا.
- إجمالي عدد ومبالغ العملاء المتأخرين.
- عرض أكثر عميل متأخر.
- فتح كشف الحساب.
- إضافة دفعة.
- إرسال تذكير واتساب.
- الانتقال إلى تعديل بيانات العميل.
- تلوين البطاقات حسب مدة التأخير.
