# ALABOUD Business Suite v13.2.1 Enterprise Professional

هذا الإصدار يصحح نظام الحوالات متعدد العملات الذي لم يكن مربوطًا بالكامل في v13.2.0.

## المزايا المنفذة فعليًا
- اختيار العملة: USD / EUR / SYP / AED / GBP / CAD.
- جلب سعر التكلفة تلقائيًا من صفحة أسعار الصرف مقابل CAD.
- سعر يدوي لهذه الحوالة فقط.
- حفظ العملة ومصدر السعر ووقت تحديثه داخل الحوالة.
- المجموع النهائي للعميل بالدولار الكندي:
  مبلغ الحوالة × سعر التحويل + الأجور.
- ربح الحوالة:
  مبلغ الحوالة × (سعر التحويل - سعر التكلفة) + الأجور.
- إخفاء سعر التكلفة والربح من الفاتورة وكشف حساب العميل.
- إظهار سعر التكلفة والربح في صفحات الإدارة والتقارير فقط.
